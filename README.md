# COPA DORADA

Aplicacion web para la gestion operativa de COPA DORADA.

## Requisitos

- Node.js 20 o superior
- npm 10 o superior

## Ejecucion local

1. Instalar dependencias:
   `npm install`
2. Ejecutar en desarrollo:
   `npm run dev`
3. Generar build de produccion:
   `npm run build`
4. Previsualizar build:
   `npm run preview`
